Track website engagement (temporary pixel)

This is a lightweight client-side tracker for GitHub Pages: it captures page_view and click events, shows a live badge, and stores events in LocalStorage. Later you can POST these to your backend and write them to Azure SQL.

External Link

Tip: Click around the page and watch the badge count increase. Use Export to download the captured JSON.

What we’re capturing

  • page_view — URL, title, referrer, session ID
  • click — tag, text snippet, id/classes, href (when applicable)
  • custom — anything with data-track will include that label
